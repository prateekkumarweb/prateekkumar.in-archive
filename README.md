# Personal website - prateekkumar.in

## Requirements

* [Node and NPM](https://nodejs.org/en/)
* [Hugo](https://gohugo.io/)

## Run

* To start the server in development mode
  ```sh
  $ hugo server
  ```
* To generate production site
  ```sh
  $ npm start
  ```
  The site is generated in `dist` folder.
